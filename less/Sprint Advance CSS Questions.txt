What is the difference between an adaptive website and a fully responsive website?
A responsive website adapts to any the device depending on the media queries. An Adaptive website will change to fit a predetermined set of sizes.



Describe what it means to be mobile first vs desktop first.
Mobile View First= Code the website to fit the mobile view first and use media queries for desktop view
Desktop View First= Code the website to fit the desktop view first and use media queries for mobile view


What does font-size: 62.5% in the html tag do for us when using rem units?
When you set the font to 62.5% it will make it so that you can set the rem by 10. In other words 16px=1.6rem. This is because 10/16=.625 which is 62.5%.
